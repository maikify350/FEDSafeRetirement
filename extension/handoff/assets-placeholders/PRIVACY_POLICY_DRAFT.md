# Privacy Policy Draft — ACT Copilot by VentureSoft

This is a starting draft only. Legal review is recommended before public/commercial rollout.

## ACT Copilot Privacy Policy

ACT Copilot by VentureSoft is a Chrome extension for approved and licensed ACT.com users. The extension helps users access VentureSoft workflow tools, templates, and productivity assistance inside supported ACT.com pages.

## Data We Collect

The extension may collect and send the following data to VentureSoft services:

- User email or account identifier.
- Extension ID and extension version.
- Browser-generated install identifier.
- Tenant/customer identifier.
- ACT database identifier or name when available.
- Supported page origin and non-sensitive page context required for licensed functionality.
- Feature usage events and error diagnostics.

## Why We Collect Data

We use this data to:

- Validate license status.
- Assign tenant-specific features.
- Enforce seat limits.
- Provide approved extension functionality.
- Troubleshoot errors.
- Protect the service from abuse.

## Data We Do Not Sell

VentureSoft does not sell extension user data and does not use extension user data for personalized, retargeted, or interest-based advertising.

## Data Transmission

Data is transmitted over HTTPS to VentureSoft backend services.

## ACT Credentials

The extension does not store ACT passwords in Chrome storage. Any ACT API credential exchange required by the service should be handled by VentureSoft backend services using secure storage and access controls.

## AI Processing

If AI-enabled features are used, user-provided text or selected context may be sent to VentureSoft-approved AI services to perform the requested action. AI features are controlled by tenant-level feature flags.

## Human Access

VentureSoft personnel do not access customer data except when necessary for support, security, legal compliance, or with customer consent.
