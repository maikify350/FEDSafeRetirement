## AI Chat System Architecture

The FEDSafe AI chat built into the CRM App uses:

- **RAG (Retrieval-Augmented Generation):** The same 7 PDFs above are chunked, embedded with OpenAI `text-embedding-3-small`, and stored in Supabase with `pgvector`
- **Matching:** When a user asks a question, the top-5 most semantically similar chunks are retrieved and injected as context
- **Model:** `gpt-4o-mini` with a federal-retirement-specialized system prompt
- **Streaming:** Real-time SSE streaming for a ChatGPT-like UX
- **Access:** Left navigation → "FEDSafe AI" → slide-in panel
